An AI Career Guidance Chatbot

An AI-powered chatbot that helps students explore suitable career paths
based on their skills, interests, and educational background.

## Features
- AI-powered career recommendations
- Retrieval-Augmented Generation (RAG)
- Simple web interface using Gradio/Streamlit
- Personalized suggestions based on user input

## Tech Stack
- Python
- LangChain
- RAG
- Gradio / Streamlit
- LLM API (Gemini/OpenAI)

## How it Works
1. User enters skills and interests
2. System retrieves relevant career information
3. LLM generates personalized suggestions
4. Chatbot displays structured career guidance

## Installation
git clone <repo>
cd Career-Guidance-Chatbot
pip install -r requirements.txt
python app.py
